Topic 1: Machine Learning Fundamentals

Understanding the core concepts, terminology, and types of machine learning

What is Machine Learning?

Definition

Machine Learning is an algorithm that optimizes a performance criterion using data of particular examples. Machines learn a selected model using training data to perform tasks autonomously.

The Core Idea

Machine learning algorithms are described as learning a target function (f) that best maps input variables (X) to an output variable (Y):

\[ Y = f(X) \]

Where:

  • X = Input variables (features/attributes)
  • Y = Output variable (target/label)
  • f = Unknown function we want to learn

We don't know what the function (f) looks like or its form. If we did, we would use it directly. Machine learning algorithms try to approximate this unknown function from data.

Types of Machine Learning

Three Main Types of Machine Learning

Supervised Learning (Focus of this module)

How Supervised Learning Works

  1. Collect and label observations (human provides labels)
  2. Split into Training (70-80%) and Test (20-30%) datasets
  3. Pass the Training dataset to the Classifier
  4. Use the model to predict classes for Test dataset
  5. Use metrics to assess results

Example: Health Prediction

Features (X): Blood Pressure, Temperature, Age, Gender, Smoking status

Label (Y): Healthy (H) or Not Healthy (NH)

The labels come from human intelligence - a supervisor who knows the correct classification.

Model Components

Classifier Components

Key Components Explained

Learning Algorithm +

A set of mathematical instructions that create a model using the training dataset. Most classifiers are named for the learning algorithm they implement (e.g., Decision Tree, Random Forest, SVM).

Loss Function (Error Function) +

Measures how far the predicted output for a single training example is from its true value. The loss function tells us how "wrong" our prediction is.

Optimizer +

An algorithm used to minimize the average loss (the Cost function) over the entire dataset. Common optimizers include Gradient Descent.

Model Parameters vs Hyperparameters +

Model Parameters: What the algorithm "learns" from data (e.g., slope and intercept of a regression line)

Hyperparameters: Tuning parameters set by the programmer that affect algorithm behavior (e.g., number of iterations, learning rate)

Predictive Modeling

There is a common principle underlying all supervised machine learning algorithms:

The Goal

Learn a target function (f) that best maps input variables (X) to an output variable (Y), allowing us to make predictions on new, unseen data.

Statistical vs Machine Learning Approach

Aspect Statistical Modeling Machine Learning
Assumptions Makes strong assumptions about data Minimal assumptions
Human Effort Requires understanding of variable relationships Minimal human effort
Process Mathematics intensive, coefficient estimation Iterative pattern discovery
Data Requirements Can work with smaller datasets Generally needs more data

Important Principle

"The fewer assumptions in a predictive model, the higher will be the predictive power."

Key Takeaways

Remember These Points

  • Machine Learning = Algorithm that learns from data to optimize performance
  • Supervised Learning uses labeled data (labels from human intelligence)
  • The goal is to learn function f where Y = f(X)
  • Classifier = Learning Algorithm + Loss Function + Optimizer
  • Model Parameters are learned; Hyperparameters are set
  • Training/Test split: typically 70-80% / 20-30%
