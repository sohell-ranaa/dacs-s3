Key Formulas Quick Reference

Accuracy

\[ Accuracy = \frac{TP + TN}{TP + TN + FP + FN} \]

Precision

\[ Precision = \frac{TP}{TP + FP} \]

Recall (Sensitivity)

\[ Recall = \frac{TP}{TP + FN} \]

F1 Score

\[ F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall} \]

Pearson Correlation

\[ \rho(X,Y) = \frac{cov(X,Y)}{\sigma_X \cdot \sigma_Y} \]

Linear Regression

\[ y = a + bX \]

Variance

\[ \sigma^2 = \frac{\sum(x_i - \bar{x})^2}{n} \]

Standard Deviation

\[ \sigma = \sqrt{\sigma^2} \]

Quick Study Checklist

Before the Test, Make Sure You Can:

  • Explain the difference between supervised and unsupervised learning
  • Describe each step of the ML pipeline
  • Calculate accuracy, precision, recall, and F1 score from a confusion matrix
  • Explain when to use different algorithms
  • Understand bias-variance tradeoff
  • Describe cross-validation and why it's used
  • Explain hyperparameter tuning methods
  • Understand resampling strategies for imbalanced data
