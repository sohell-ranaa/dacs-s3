Essential Formulas

Accuracy

\[ \frac{TP + TN}{TP + TN + FP + FN} \]

Precision

\[ \frac{TP}{TP + FP} \]

Recall (Sensitivity)

\[ \frac{TP}{TP + FN} \]

F1 Score

\[ 2 \times \frac{Precision \times Recall}{Precision + Recall} \]

Pearson Correlation

\[ \rho(X,Y) = \frac{cov(X,Y)}{\sigma_X \cdot \sigma_Y} \]

Variance

\[ \sigma^2 = \frac{\sum(x_i - \bar{x})^2}{n} \]

Standard Deviation

\[ \sigma = \sqrt{\sigma^2} \]

Linear Regression

\[ y = a + bX \]

Key Concepts Quick List

Supervised Learning

Uses labeled data. Y = f(X). Labels from humans.

Bias vs Variance

High Bias = Underfit. High Variance = Overfit.

K-Fold CV

Train on k-1 folds, test on 1. Repeat k times.

One-Hot Encoding

Convert categorical to binary columns. pandas.get_dummies()

StandardScaler

Mean=0, Std=1. Fit on train only!

MinMaxScaler

Values between 0-1. Fit on train only!

ROC Curve

TPR vs FPR. Top-left = perfect. AUC summarizes.

SMOTE

Creates synthetic minority samples. Better than duplication.

Grid Search CV

Tests all hyperparameter combinations with CV.

McNemar's Test

Compares model predictions. p < 0.05 = significant.

Support Vectors

Points on margin defining SVM hyperplane.

Bagging vs Boosting

Bagging = parallel. Boosting = sequential.

Data Preparation Checklist

  • Missing Values: Delete if >30%, else impute (mean/median/mode)
  • Categorical: One-Hot Encoding (pandas.get_dummies)
  • Scaling: Required for gradient descent and distance-based algorithms
  • Split: 70-80% Train / 20-30% Test
  • Data Leakage: Fit scaler on TRAIN only, transform both
