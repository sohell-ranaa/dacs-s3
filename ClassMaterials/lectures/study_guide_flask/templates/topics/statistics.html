Topic 2: Statistics & Model Logic

Statistical concepts, measures of central tendency, and scientific reasoning

Why Do We Need Statistics?

Quote to Remember

"Impossible things usually don't happen." - Sam Treiman, Princeton University

Statistics helps us quantify "usually."

A statistic is "a quantity that is computed from a sample of data." Statistics help us:

  • Draw conclusions from data
  • Summarize in a meaningful way in presence of noise
  • Make probabilistic statements about populations

Measures of Central Tendency

These summarize data with a single number. The three most popular are:

Example

Dataset: 2, 3, 3, 4, 5, 5, 5, 7, 9

  • Mean: (2+3+3+4+5+5+5+7+9) / 9 = 43/9 = 4.78
  • Median: 5 (middle value when sorted)
  • Mode: 5 (appears most often - 3 times)

Distribution Types

Distribution Characteristics Mean vs Median vs Mode
Normal (Gaussian) Symmetric bell curve Mean = Median = Mode
Bimodal Two peaks Two modes
Positively Skewed Tail extends right Mode < Median < Mean
Negatively Skewed Tail extends left Mean < Median < Mode

Measures of Variability

Why Variability Matters

Summarizing by a single number (mean) is rarely enough. If two systems have the same mean, we tend to prefer the one with less variability.

Variance

\[ \sigma^2 = \frac{\sum(x_i - \bar{x})^2}{n} \]

Variance is the average of squared distances from the mean.

Problem: Units are squared (changing units changes the answer squared).

Standard Deviation

\[ \sigma = \sqrt{\sigma^2} = \sqrt{\frac{\sum(x_i - \bar{x})^2}{n}} \]

Same unit as mean, so we can compare to mean.

Coefficient of Variation (C.O.V.)

\[ C.O.V. = \frac{\sigma}{\mu} \]

Takes units out and shows relative magnitude of variation.

Correlation and Covariance

Pearson Correlation Coefficient

\[ \rho(X,Y) = \frac{cov(X,Y)}{\sigma_X \cdot \sigma_Y} \]

Normalized covariance (ranges from -1 to +1):

  • +1: Perfect positive linear relationship
  • 0: No linear association (but may still be dependent!)
  • -1: Perfect negative linear relationship

Important!

A correlation of zero suggests no linear association, but does NOT mean two variables are independent! There may be non-linear relationships.

Scientific Reasoning Methods

Method Direction Description Certainty
Deductive General → Specific If A and B, then always C Guaranteed if premises true
Inductive Specific → General If A and B, then most probably C Probable, not guaranteed
Abductive Effect → Cause If A and B, then C maybe? Worthy of investigation

Statistical Modeling vs Predictive Modeling

Test Tip

The lecturer emphasized understanding these definitions for the test.

AspectStatistical ModelingPredictive Modeling
Definition Formalizes relationship between variables in the form of mathematical equations Learns a target function that maps input to output, then makes predictions on new data
Formula Explicitly defined by user (e.g., y = ax + b) Learned from data by algorithm
Coefficients Found using mathematical methods Learned automatically from training
Purpose Understand relationships Make predictions on new data

Predictive Modeling - Two Steps

  1. Model the relationship between input (X) and output (Y) using machine learning
  2. Make predictions for new values of Y when given new values of X

The purpose of machine learning IS predictive modeling!

Key Takeaways

Remember These Points

  • Mean, Median, Mode = indices of central tendency
  • Variance and Standard Deviation = measures of variability
  • Pearson correlation ranges from -1 to +1
  • Correlation = 0 doesn't mean independence!
  • C.O.V. = σ/μ (removes units for comparison)
  • Deductive (general→specific), Inductive (specific→general), Abductive (suggests hypothesis)
